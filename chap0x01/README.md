#### (2)实验步骤

------

##### 1.提前下载好纯净版 Ubuntu 安装镜像 iso 文件

![下镜像](/jpg/镜像.jpg)

##### 2.“有人值守”方式安装好一个可用的Ubuntu系统环境。

(1)创建一个新的虚拟机

![创建虚拟机](/jpg/ubuntu设置1.jpg)

(2)为该虚拟机设置好网卡

![设置网卡](/jpg/ubuntu设置2.jpg)

(3)导入提前下载好的镜像

![导入镜像](/jpg/ubuntu设置3.jpg)

(4)启动该虚拟机

![启动虚拟机](/jpg/ubuntu设置4.jpg)

(5)对虚拟机进行一系列设置

![虚拟机设置](/jpg/ubuntu设置5.jpg)

![虚拟机设置](/jpg/ubuntu设置6.jpg)

(7)进行个性化设置

![用户名密码设置](/jpg/ubuntu设置7.jpg)

(8)等待系统自动安装

##### 3.制作包含 user-data 和 meta-data 的 ISO 镜像文件，假设命名为 focal-init.iso

(1)在cmd里面进行一系列操作之后得到user-data和meta-data文件

![终端操作](/jpg/终端设置1.jpg)

(2)将user-data和meta-data下载到本地

![下载文件](/jpg/下载文件.jpg)

(3)利用老师提供网址制作iso镜像

##### 4.移除上述虚拟机「设置」-「存储」-「控制器：IDE」

![移除](/jpg/镜像挂载1.png)

##### 5.在「控制器：SATA」下新建 2 个虚拟光盘，按顺序 先挂载「纯净版 Ubuntu 安装镜像文件」后挂载 focal-init.iso

![挂载镜像](/jpg/镜像挂载1.png)

##### 6.启动虚拟机，稍等片刻会看到命令行中出现以下提示信息。此时，需要输入 yes 并按下回车键，剩下的就交给「无人值守安装」程序自动完成系统安装和重启进入系统可用状态了

##### 7.将该虚拟机改为多重加载模式。

![多重加载](/jpg/多重加载.jpg)
